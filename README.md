# StarrySky Theme

StarrySky Theme for Visual Studio 2022/2026

## Screenshots

![C#](./StarrySky-ss.png)

## Installation

1. Download the extension (vsix) file
1. Install the extension to Visual Studio 2022 or 2026
1. Select Tools > Theme > StarrySky
